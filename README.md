Tool Tracker

Tool Tracker is a lightweight, zero-backend web app for managing tool check-ins and check-outs. It runs entirely in your browser using localStorage, making it perfect for small teams, workshops, or job sites that need to keep track of who has what â€” without setting up a database or server.

Features:

Add, edit, and archive tools easily
Check out tools to a job and person 
Check tools back in with condition and notes

Search and filter by status, category, or job
Export and import data as JSON backups
Clean, responsive UI built with vanilla JavaScript

All data stored locally in your browser
Construction crews tracking shared equipment
Maker spaces or fab labs loaning tools
Small shops managing repair instruments
Field teams logging borrowed devices
QR Code and RFID functionality to Simplify Sign in and out
